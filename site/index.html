About me

Welcome to my slice of the internet! I'm Matt, a.k.a the Schwenckenator, an Aussie front-end web dev by day, and working on whatever catches my fancy by night.
I'm interested in Game dev, Web dev, Table-top RPGs, and Music, not necessarily in that order.

I have built this site from scratch, currently using a custom python static site builder script. You are welcome to check out the repo here, but beware, it's very particular to how my site is made. Rather than using my script as is, I would instead recommend learning from it and writing your own.

I tend to write about any projects that I am working on, with occasional bouts of rambling nonsense on extremely niche and petty topics. See below for my latest!
